Lab: Package Management and System Updates

Description:
This lab teaches you how to manage Ubuntu packages exclusively using the command line.
You will learn to install, remove, and update software using apt, dpkg, and snap,
work with PPAs, and configure automatic security updates.
You will also explore what each software package is for and test its functionality
before and after removal.

============================================================

Exercise 2: Add and Remove a PPA for CLI Tools

Objective:
Learn how to add and remove Personal Package Archives (PPAs) that provide CLI tools.

Steps

- [ ] Install the PPA utility:
  Command: `sudo apt install software-properties-common -y`
  Explanation: Installs tools required to add and remove PPAs.

- [ ] Add the Fastfetch PPA (Fastfetch displays system info as ASCII art):
  Command: `sudo add-apt-repository ppa:zhangsongcui3371/fastfetch`

- [ ] Update the package index:
  Command: `sudo apt update`

- [ ] Install Fastfetch:
  Command: `sudo apt install fastfetch`

- [ ] Run Fastfetch to view system specs:
  Command: `fastfetch`

- [ ] Remove Fastfetch:
  Command: `sudo apt remove fastfetch -y`

- [ ] Verify removal:
  Command: `fastfetch`
  Expected result: command not found

- [ ] Remove the PPA:
  Command: `sudo add-apt-repository --remove ppa:zhangsongcui3371/fastfetch -y`

- [ ] Explore the manual page:
  Command: `man add-apt-repository`
