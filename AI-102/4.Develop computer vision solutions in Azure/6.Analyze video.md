# Azure Video Indexer

## Overview
- Azure Video Indexer extracts **insights from video files**
- Designed for:
  - Analysis
  - Indexing
  - Searchability
- Works through:
  - Web portal
  - Widgets
  - REST APIs

## Common Use Cases
- Meeting and call recordings
- Training and learning videos
- Media archives
- Surveillance and event footage
- Content moderation and compliance

## Core Azure Video Indexer Capabilities
Azure Video Indexer can automatically extract:

### Visual Insights
- Facial recognition (Limited Access required)
- Optical Character Recognition (OCR)
- Scene segmentation (scene and shot boundaries)
- Labels (objects, themes, activities)
- Content moderation (adult / violent content)

### Audio & Language Insights
- Speech transcription
- Topics discussed in the video
- Sentiment analysis (positive / neutral / negative segments)
- Keyword and phrase extraction

## Video Indexer Portal
- Web-based portal for:
  - Uploading videos
  - Viewing indexed insights
  - Managing custom models
- Automatically indexes videos after upload

## Video Indexing Workflow
1. Upload video to Azure Video Indexer
2. Video is processed and indexed
3. Insights are extracted automatically
4. Results are viewable via:
   - Portal
   - Widgets
   - APIs

## Custom Insights
Azure Video Indexer includes prebuilt models, but supports **custom models** for advanced scenarios.

### Custom People Models
- Train recognition for specific individuals
- Requires:
  - Images of faces
  - Limited Access approval
- Used for:
  - Identifying known people across videos

### Custom Language Models
- Improve transcription accuracy
- Useful when:
  - Industry-specific terms are used
  - Acronyms or internal jargon are common

### Custom Brands Models
- Train recognition for:
  - Company names
  - Products
  - Projects
- Useful for:
  - Conference recordings
  - Marketing and brand analysis

## Using Azure Video Indexer in Applications
There are two main integration approaches:

## Azure Video Indexer Widgets
- Prebuilt UI components
- Can be embedded in:
  - Custom web pages
  - Internal portals
- Allows:
  - Video playback
  - Insight visualization
- Does NOT require portal access for viewers

## Azure Video Indexer REST API
- Used for automation and integration
- Common capabilities:
  - Upload and index videos
  - Retrieve insights
  - Manage custom models
  - List and manage videos
- Requires:
  - Account access token

### Access Token Endpoint
- Used to retrieve a token before calling APIs
- Token required for all REST API calls

## Example API Use Cases
- List videos in an account
- Retrieve indexed insights
- Create or delete custom models
- Automate large-scale video processing

## Deployment Options
- Azure Video Indexer resource
- Can be deployed using:
  - Azure Portal
  - ARM templates

## Key Exam Concepts
- Video must be **uploaded and indexed** before analysis
- Video Indexer ≠ Azure Vision Image Analysis
- Custom people recognition requires **Limited Access**
- Widgets = UI embedding
- APIs = automation and programmatic access
