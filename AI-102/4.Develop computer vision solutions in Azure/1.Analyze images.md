# Azure Vision – Image Analysis

## Overview
- Azure Vision Image Analysis extracts insights from images.
- Common use cases:
  - Generate captions
  - Suggest tags
  - Detect objects
  - Detect people
  - Extract text (OCR)

## Core Image Analysis Capabilities
- Caption generation (natural language description)
- Tag suggestion (objects, scenery, actions)
- Object detection (with bounding boxes)
- People detection (with bounding boxes)
- Dense captions (multiple localized descriptions)
- OCR (read printed or handwritten text)
- Smart crops (area of interest)

## Supported Image Requirements
- Formats:
  - JPEG
  - PNG
  - GIF
  - BMP
- File size:
  - Less than 4 MB
- Dimensions:
  - Greater than 50 x 50 pixels

## Provisioning Options
- Microsoft Foundry project + Foundry Tools (recommended)
- Foundry Tools standalone resource
- Standalone Azure Vision resource
  - Includes a free tier for experimentation

## Connecting to Azure Vision
- Endpoint format:
  https://<resource_name>.cognitiveservices.azure.com/
- Authentication methods:
  - Key-based authentication
  - Microsoft Entra ID authentication
- Production best practices:
  - Managed identity (Entra ID)
  - Azure Key Vault for keys

## Analyze Image API
- REST API or SDK (Python / .NET)
- Submit image as:
  - Binary image data
  - Image URL

## Python SDK – Basic Pattern
- Create ImageAnalysisClient
- Call analyze() or analyze_from_url()
- Specify desired visual features

## VisualFeatures Enumeration
- TAGS
  - Identifies objects, scenery, actions
- OBJECTS
  - Detects objects with bounding boxes
- CAPTION
  - Generates a single natural language caption
- DENSE_CAPTIONS
  - Generates multiple localized captions
- PEOPLE
  - Detects people with bounding boxes
- SMART_CROPS
  - Suggests crops for specific aspect ratios
- READ
  - Extracts readable text (OCR)

## Response Characteristics
- Confidence scores for tags and captions
- Bounding boxes for:
  - Objects
  - People
  - Dense captions
- Metadata includes:
  - Image width
  - Image height

## Example Response Elements
- apim-request-id
- modelVersion
- denseCaptionsResult
- boundingBox (x, y, w, h)
- confidence scores
- metadata
